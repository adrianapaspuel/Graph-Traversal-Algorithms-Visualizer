# Graph Traversal Algorithms Visualizer

An interactive visual representation of the pathfinding algorithms built with HTML, CSS and JavaScript.

## Implemented Algorithms
- Dijktra's Algorithm
- Breadth First Search (BFS)
- Depth First Search (DFS)
- A* Search

## Features
- Interactive grid
- Animated algorithm execution
- Path visualization
- Reset grid functionality
- Run and stop functionality
- Start and end node selection
- Wall creation

## Technologies

- HTML
- CSS
- JavaScript

## Future Improvements
- Finish polishing A* Search
- Weighted Nodes
- Performance comparison
